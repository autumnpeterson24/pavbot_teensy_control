DEMO YIPPEE!! :D

TERMINAL 1
1. Enter ROS envirnoment (i.e. cd ros2_ws)
2. cd ~/ros2_ws
    source /opt/ros/humble/setup.bash
    source install/setup.bash
2a. source /opt/ros/jazzy/setup.bash
    source install/setup.bash
3. launch the nodes:
    ros2 launch pavbot_teensy_control teensy_control.launch.py

TERMINAL 2
1. watch WheelCmd 
    source /opt/ros/humble/setup.bash
    source ~/ros2_ws/install/setup.bash
    ros2 topic echo /teensy/wheels
1a. watch WheelCmd 
    source /opt/ros/jazzy/setup.bash
    source ~/ros2_ws/install/setup.bash
    ros2 topic echo /teensy/wheels

TERMINAL 3
1. watch transport state
    source /opt/ros/humble/setup.bash
    source ~/ros2_ws/install/setup.bash
    ros2 topic echo /teensy/transport_state
1a. watch transport state
    source /opt/ros/jazzy/setup.bash
    source ~/ros2_ws/install/setup.bash
    ros2 topic echo /teensy/transport_state

TERMINAL 4
1. simulate Nav2 publishing /cmd_vel

STRAIGHT FORWARD
    ros2 topic pub -r 10 /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.4}, angular: {z: 0.0}}"

TURN IN PLACE
    ros2 topic pub -r 10 /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.0}, angular: {z: 0.8}}"

CURVE (forward+turn)
    ros2 topic pub -r 10 /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.3}, angular: {z: 0.5}}"
